# Конвертер валют (ЦБ РФ) — Streamlit

Простое веб‑приложение на Streamlit для конвертации валют с использованием актуальных курсов Центрального Банка РФ со страницы `https://cbr.ru/currency_base/daily/`.

## Возможности
- Получение актуальных курсов с сайта ЦБ РФ
- Выбор исходной и целевой валюты
- Конвертация с учётом номинала (единиц) валюты
- Обработка ошибок сети/парсинга и неверного ввода
- Кэширование результатов с возможностью принудительного обновления

## Архитектура
- `data_fetcher.py` — загрузка страницы ЦБ и парсинг таблицы в `pandas.DataFrame`, нормализация, добавление `RUB`.
- `converter.py` — функции конвертации между валютами по формуле: `amount * (Value_A/Nominal_A) / (Value_B/Nominal_B)`.
- `app.py` — интерфейс Streamlit: ввод суммы, выбор валют, показ результата, кнопка «Обновить курсы».

## Требования
См. `requirements.txt` (основные: `streamlit`, `pandas`, `requests`, `beautifulsoup4`, `lxml`).

## Установка и запуск
Рекомендуется использовать виртуальное окружение Python 3.10+

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate  # Windows PowerShell

pip install -r requirements.txt
streamlit run app.py
```

После запуска откроется локальный адрес, например `http://localhost:8501`.

## Использование
1. Введите сумму для конвертации.
2. Выберите исходную и целевую валюты.
3. Нажмите «Конвертировать» — результат отобразится ниже.
4. При необходимости нажмите «Обновить курсы», чтобы принудительно обновить данные (кэш автоматически обновляется раз в час).

## Примечания
- Данные курсов загружаются из официальной таблицы ЦБ РФ. Столбец «Курс» публикуется за фиксированное число единиц валюты («Единиц»), поэтому при конвертации учитывается номинал: `Value/Nominal` — цена одной единицы в RUB.
- При недоступности источника или изменении структуры страницы вы увидите сообщение об ошибке в интерфейсе.
